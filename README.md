„Cinema Management System – C++ Client-Server Application”.

Kluczowe Technologie: C++20, PostgreSQL (libpqxx), Multithreading (mutex), OOP.

Funkcje (Features):

Podział na role: Administrator (zarządzanie bazą) i Klient (rezerwacje).

Zabezpieczenie przed jednoczesnym dostępem (Thread-safe database access).

Logika biznesowa: obsługa miejsc VIP, walidacja technologii sal (3D/4D)
